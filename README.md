## Clearn

This is a simple tool to learn c++ with cmake.

## Prerequisites
- CMake >= 3.15
- C++ compiler with C++14 support
- OpenSSL 3 and SQLite3 libraries
- Visual Studio Code (Extensions: C/C++, CMakeTools by Microsoft)

## Build

```bash
mkdir build
cd build
cmake ..
make
```

## Run

```bash
cd build
./bin/clearn
```
